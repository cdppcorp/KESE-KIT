# Drafts — 초안 폴더

## 구조
각 장별로 폴더가 생성됩니다.

```
drafts/
├── ch01/          # 1장 개요
├── ch02/          # 2장 법적 기반과 프레임워크
├── ch03/          # 3장 기술적 취약점 분석
├── ch04/          # 4장 관리적 취약점 분석
├── ch05/          # 5장 물리적 취약점 분석
├── ch06/          # 6장 실무 적용 가이드
├── ch07/          # 7장 자동화 도구 활용
└── appendix/      # 부록
```

## 파일 명명 규칙
- `old.md` — 원본/이전 버전
- `new.md` — 퇴고/수정 버전
- `section-N.md` — 절별 초안
